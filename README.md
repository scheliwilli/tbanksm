Планировщик маршрута, который объединяет в одном поиске железнодорожные, авиамаршруты и автобусные перевозки. Система анализирует доступные варианты перемещения между городами, строит комбинированные маршруты и помогает пользователю выбрать оптимальный вариант по стоимости, времени или количеству пересадок.

Технологический стек
Frontend:
       React — пользовательский интерфейс
Backend:
       Python — язык разработки
       FastAPI — веб-фреймворк для создания REST API
