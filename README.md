       Сценарий использования
Один продукт - несколько сценариев
Ряд вопросов:
1. Кто пользователь?      - Человек которые отправляется в отпуск командировку
2. Что он хочет сделать?       - Увидеть оптимальный список возможных способов добраться
3. Что он вводит/выбирает?    - Пункт начала, пункт конца, дату отправления 

предпочение по сортировке из списка:
    По стоимости,
    По длительности 
    По близости времени
    По количеству пересадок

Фильтры:
    Средство передвижения

4. Что делает система?    - Выявляет список оптимальных путей по запросу клиента
5. Что пользователь получает?    - Для каждого пути последовательность рейсов, сумарная цена, сумарное затраченое время, ссылки на ресурс, расчитывать время между рейсами
6. Что моглоп пойти не так?    - 
Дата будущая
Существующий город
Наличие билетов

]
}
